# AncientCrafts

This is the AncientCrafts Android project. The repository on GitHub is:

https://github.com/Reym-Tech/AncientCrafts

How to build locally

Prerequisites:
- JDK 17+
- Android SDK and Android Studio (or command-line Gradle)

Build from command line:
```bash
./gradlew assembleDebug
```
